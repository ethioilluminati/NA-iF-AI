
# NA’iF AI – Your Superhuman Full Stack Engineer

## How to Run

1. Install Node.js and npm
2. Run: `npm install next react react-dom`
3. Start the dev server: `npx next dev`
4. Open browser: http://localhost:3000

## Project Structure

- `pages/index.js` — Main chat interface
- `pages/api/generate.js` — Backend API to simulate AI-generated code
