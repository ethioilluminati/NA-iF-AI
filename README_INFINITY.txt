# 🌌 NA’iF AI - Infinity Mode

## Features
- 🔐 Firebase Authentication (Google Login)
- 💳 Stripe Payments (Premium Access)
- ⚙️ AI Chat App Builder
- 📊 User Dashboard

## Setup
1. Add your Firebase config in `firebaseConfig.js`
2. Connect your Stripe keys
3. Run the app:
```bash
npm install
npm run dev
```

## GitHub Upload
```bash
git init
```
